# Juego Adivinar Numero

Un juego simple de consola en Java donde debes adivinar un numero secreto.

## Descripcion

El programa genera aleatoriamente un numero entre 1 y 50, y tienes hasta 7 intentos para adivinarlo. Despues de cada intento, recibiras pistas indicando si tu numero es mayor o menor que el numero secreto.

## Caracteristicas

- Numero aleatorio entre 1 y 50
- Maximo 7 intentos
- Pistas de "mas alto" o "mas bajo"
- Estadisticas al finalizar la partida

## Requisitos

- Java 8 o superior
- JDK instalado en tu sistema

## Ejemplo de uso
```
=== JUEGO: ADIVINA EL NUMERO ===
He pensado un numero entre 1 y 50
Puedes adivinarlo?

Intento 1/7 - Tu numero: 25
Demasiado bajo. Intenta con un numero mas alto.

Intento 2/7 - Tu numero: 40
Demasiado alto. Intenta con un numero mas bajo.

Intento 3/7 - Tu numero: 33
FELICIDADES! Acertaste en 3 intentos!

--- Estadisticas ---
* * * 
Usaste 3 de 7 intentos
```
