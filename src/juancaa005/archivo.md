# Organización del repositorio

## 📌 Rama personal

He creado una rama con mi nombre para trabajar de forma independiente sobre mis cambios y evitar afectar directamente a la rama principal.

Esta rama se utilizará para:
- Desarrollo de nuevas funcionalidades
- Pruebas
- Cambios personales antes de ser revisados o fusionados

## 📁 Carpeta `src`

Dentro del proyecto he creado la carpeta **`src`**, que será el directorio principal donde se almacenarán los archivos del código fuente.

En esta carpeta se incluirán:
- Archivos de lógica principal
- Recursos necesarios para el funcionamiento del proyecto
- Código organizado de forma clara y estructurada

## ✅ Objetivo

Mantener el proyecto ordenado, escalable y fácil de mantener, separando correctamente el código fuente y trabajando mediante ramas.
