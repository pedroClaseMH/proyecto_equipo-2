# Carpeta lib

En esta carpeta se incluyen **librerías o recursos externos** necesarios para el proyecto.

## Qué puede ir aquí
- Librerías externas
- Archivos de apoyo al proyecto

## Qué NO debe ir aquí
- Código desarrollado por el equipo
- Archivos personales
- Versiones compiladas

> Si no se utilizan librerías externas, esta carpeta puede permanecer vacía.
